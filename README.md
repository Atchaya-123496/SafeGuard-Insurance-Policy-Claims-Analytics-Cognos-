# 🛡️ SafeGuard Insurance – Policy & Claims Analytics Dashboard

**IBM Cognos Style | Interactive Charts | Real-time Filters**

A professional, fully responsive insurance analytics dashboard with animated login, dynamic KPIs, and interactive filtering by **Agent Branch** (Chennai, Bangalore, Hyderabad) and **Policy Type** (Health, Motor, Life, Property). Built with HTML5, CSS3, JavaScript, and Chart.js.

![Dashboard Preview](https://via.placeholder.com/800x450?text=SafeGuard+Dashboard+Preview)

## ✨ Features

- 🔐 **Secure Login Page** – Glass morphism design with animation
- 📊 **Live KPIs** – Total Policies, Current Claim ID, Avg Settlement Days, Fraud Cases Flagged
- 🥧 **Claim Status Distribution** – Donut chart (Rejected / Pending / Settled)
- 📈 **Claims by Policy Type** – Bar chart (Health, Motor, Life, Property)
- 📉 **Monthly Claim Trend** – Line chart (Jan–Dec 2025)
- 🚨 **Fraud Breakdown** – Theft, Accident, Forgery, Other
- 🎛️ **Dynamic Filters** – Change Agent Branch or Policy Type → all charts & KPIs update instantly
- 🌟 **Animated Background** – Floating particles, smooth transitions
- 📱 **Fully Responsive** – Works on desktop, tablet, and mobile
